React Hooks – Study Notes

These notes are for revision and self-study. They summarize the purpose, rules, and common use cases of React Hooks.

useState

useState adds local state to a function component.

  • State changes trigger re-renders
  • State updates are asynchronous
  • State belongs to the component
function Counter() {
  const [count, setCount] = React.useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      {count}
    </button>
  );
}
Do not mutate state directly.

useEffect

useEffect runs side effects after rendering.

  • Fetching data
  • Subscriptions
  • Timers
React.useEffect(() => {
  fetch("/api/users")
    .then(res => res.json())
    .then(data => setUsers(data));
}, []);
Dependency array controls when the effect runs.

useMemo

useMemo memoizes a calculated value.

  • Use for derived data
  • Use for expensive calculations
  • Not needed for simple values
const activeUsers = React.useMemo(() => {
  return users.filter(u => u.active);
}, [users]);

useCallback

useCallback memoizes a function reference.

  • Useful with memoized child components
  • Useful in dependency arrays
const increment = React.useCallback(() => {
  setCount(c => c + 1);
}, []);

useRef

useRef stores mutable values without causing re-renders.

  • DOM access
  • Timers
  • Persistent values
const inputRef = React.useRef(null);

inputRef.current.focus();

Combined Example (Reference)

Example showing common hooks used together.

function UserSearch() {
  const [query, setQuery] = React.useState("");
  const [users, setUsers] = React.useState([]);
  const inputRef = React.useRef(null);

  React.useEffect(() => {
    if (!query) return;

    fetch("/api/users?q=" + query)
      .then(res => res.json())
      .then(data => setUsers(data));
  }, [query]);

  const activeUsers = React.useMemo(() => {
    return users.filter(u => u.active);
  }, [users]);

  return (
    <div>
      <input
        ref={inputRef}
        value={query}
        onChange={e => setQuery(e.target.value)}
      />

      <ul>
        {activeUsers.map(u => (
          <li key={u.id}>{u.name}</li>
        ))}
      </ul>
    </div>
  );
}
